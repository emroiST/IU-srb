# 🌱 Industrial Upgrade — Справочник семян

Интерактивный веб-справочник по селекции растений для мода **Industrial Upgrade 1.21.1**.

## 🔗 Ссылки

- **GitHub Gist**: [открыть онлайн](https://emroist.github.io/IU-srb/)

## ✨ Возможности

- 🔍 **Поиск** — по названию или ID растения
- 🏷️ **Категории** — Базовые, Растения, Металлы, Редкие, Радиоактивные, Эндер/Незер, Ядерные
- 🌳 **Дерево рецептов** — полная цепочка создания с вложенностью
- 📱 **Адаптивный дизайн** — работает на ПК и мобильных

## 📖 Категории

| Категория | Описание |
|-----------|----------|
| 🌿 Базовые | 16 растений, не требующих скрещивания |
| 🌾 Растения | Сельскохозяйственные культуры |
| ⚙️ Металлы | Медь, железо, золото, алмаз и др. |
| 💎 Редкие | Титан, хром, платина, иридий и др. |
| ☢️ Радиоактивные | Уран, плутоний, молибден и тяжёлые металлы |
| 🔮 Эндер/Незер | Растения из Незера и Энда |
| 🧬 Ядерные | Финальные растения (Америций, Нептуний, Кюрий) |


## 🎮 Для чего это нужно

Мод **Industrial Upgrade** добавляет сложную систему селекции растений для получения металлических и ядерных ресурсов. Этот справочник помогает:

- Быстро найти рецепт нужного растения
- Увидеть полную цепочку создания (от базовых до финальных)
- Спланировать развитие в игре
